
# ForumJob Server

ForumJob Server is the backend API for the ForumJob platform.  
It provides authentication, forum discussion, job vacancy management, company management, user profile, admin dashboard, import, and export features for the ForumJob ecosystem.

## Features

- User registration and login
- JWT-based authentication
- User profile management
- Password change endpoint
- Forum post management
- Forum comment management
- Job vacancy management
- Company data management
- Admin dashboard data
- User management
- Job import and export
- Company import and export
- Role-based admin access

## Tech Stack

- PHP
- Laravel 8
- JWT Auth
- Laravel Sanctum
- Laravel CORS
- Cloudinary Laravel
- Laravel Excel
- MySQL or compatible relational database

## Requirements

- PHP 7.3 or newer
- Composer
- MySQL or MariaDB
- Laravel supported local server
- Node.js and npm/yarn, if frontend assets are needed
